(02.)

RESPOSTA: Usa-se roteamento estático quando se tem uma rede com um número limitado de roteadores para outras redes. Para dinâmico é com redes com mais de uma rota possível para o mesmo ponto. Uma tabela de roteamento estático é construída manualmente pelo administrador do sistema, podendo ou não ser divulgada para outros dispositivos de roteamento na rede. E essas tabelas não são ajustadas automaticamente a um mudança da rede. Já a tabela de roteamento dinamico é formada a partir de dados trocadas entre protocolos de roteamento. Os protocolos são projetados para distribuir informações que ajustam rotas dinamicamente em relação as alterações nas condições da rede.

RESPOSTA: É um dos algoritmos que calcula a rota de custo mínimo entre vértices de um grafo. Selecionado um vértice como raiz da busca, este algoritmo calcula o custo mínimo deste vértice para todos os outros vértices do grafo.
